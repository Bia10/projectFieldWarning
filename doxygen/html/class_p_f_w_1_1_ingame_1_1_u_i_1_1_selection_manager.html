Detailed Description

Responsible for the set of selected units.

Tracks which units are currently selected, adds and removes units from the selection, dispatches orders to the selected units.

Send a movement command to the currently selected platoons.

Parameters
useGhostHeadingIf true, the platoons will move to their sillhouettes (e.g. the command was previewed using mouse drag and the units should move to the positions that were shown in the preview). If false, the platoons will just pick their destinations based on where the cursor is.
